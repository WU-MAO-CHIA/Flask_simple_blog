# Flask-blog-Demo
這是一個小小的 Flask 實作，建立一個blog(參考[為你自己學 PYTHON](https://pythonbook.cc/chapters/flask/how-website-works))
透過 Flask 進行全棧開發

後續透過 docker 將 flask, ngnix, gunicorn 打包成為 container，解決 gunicorn無法跑在 window 上的問題

## docker build
```docker
docker compose build 
```

## docker run
```
docker compose up 
```
